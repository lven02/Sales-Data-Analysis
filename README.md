# Sales-Data-Analysis
## Some Data Analysis using SQL and Power BI exploring a sales dataset found from Kaggle: https://www.kaggle.com/datasets/vinothkannaece/sales-dataset
The original values in Sales_Amount column of the dataset made little sense so I recalculated them using other values in the table for better, more realistic analysis. I also added columns for the cost of each transaction (how much those units cost the company) and the amount lost through discounts.  
The Power BI screenshots show that the company is losing money due to the large amount of discounts being offered, as if these were even slashed in half then the profit would be in the green. Overall, the sales performed in this dataset are not profitable.  
Other interesting takeaways are that David is the top sales rep and Charlie is the bottom one, meaning Charlie may need some retraining or extra motivation. Food is seen as the lowest selling sector.
